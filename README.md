## <a name="introduction">ğŸ¤– Introduction</a>

Mann ki Baat is an advanced AI-driven SaaS platform that empowers users to create, discover, and enjoy podcasts with unparalleled ease and sophistication. At its core, the platform leverages cutting-edge text-to-audio conversion technology enhanced by multi-voice AI capabilities, ensuring every podcast maintains exceptional quality and authenticity. Seamless playback across devices enriches the listening experience, while intuitive tools for podcast thumbnail image generation add visual appeal and branding.

Beyond its technological prowess, Mann ki Baat fosters a collaborative community environment where creators can refine their content, receive feedback, and connect with like-minded individuals. With customizable themes and user-friendly interfaces, the platform supports creators at every stage, from novice to expert, in crafting compelling audio narratives that resonate globally. Mann ki Baat is not just redefining podcast creation; it's shaping the future of audio storytelling, where innovation meets creativity to inspire and engage audiences worldwide.

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

<p align="left">
  <img src="https://www.animatedimages.org/data/media/562/animated-line-image-0184.gif" width="1920" 
</p>

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone https://github.com/mrmayankmathur/Mann-ki-Baat.git
cd Mann-ki-Baat
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Set Up Environment Variables**

Create a new file named `.env` in the root of your project and add the following content:

```env
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_URL='/sign-in'
NEXT_PUBLIC_CLERK_SIGN_UP_URL='/sign-up'
```

Replace the placeholder values with your actual Convex & Clerk credentials. You can obtain these credentials by signing up on the [Convex](https://www.convex.dev/) and [Clerk](https://clerk.com/) websites.

**Running the Project**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.
